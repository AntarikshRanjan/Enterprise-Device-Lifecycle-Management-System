# AssetFlow - Enterprise Device Lifecycle Management System

A production-grade, full-stack enterprise web application for managing the lifecycle of organizational IT assets including Laptops, Monitors, Phones, Keyboards, and Accessories.

## Tech Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, shadcn/ui, React Router, TanStack Query, React Hook Form, Zod
- **Backend**: Node.js, Express, TypeScript, Prisma ORM, PostgreSQL, JWT
- **Deployment**: Vercel (Frontend), Railway (Backend), Neon (Database)

## Architecture
This project uses a monorepo structure containing both the frontend and backend applications, strictly adhering to Clean Architecture and SOLID principles.
